# FutureBuilder Demo - El Constructor del Futuro

Demo educativa sobre programación asíncrona en Flutter usando FutureBuilder.

## Capturas

- Estado Loading: Spinner mientras carga datos
- Estado Success: Lista de usuarios
- Estado Error: Manejo de errores con botón de reintentar

## Cómo ejecutar
```bash
# Clonar el repositorio
git clone https://github.com/(TU-USUARIO)/futurebuilder-demo.git

# Entrar a la carpeta
cd futurebuilder-demo

# Instalar dependencias
flutter pub get

# Ejecutar
flutter run -d chrome
```

## Conceptos clave

- **Future<T>**: Promesa de un valor futuro
- **FutureBuilder**: Widget que construye UI basándose en el estado de un Future
- **ConnectionState**: Estados del Future (none, waiting, active, done)
- **AsyncSnapshot**: Contiene el estado y datos actuales
